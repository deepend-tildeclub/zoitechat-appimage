# ZoiteChat

ZoiteChat is an IRC client for Windows and UNIX-like operating systems.  
See [IRCHelp.org](http://irchelp.org) for information about IRC in general.  
For more information on ZoiteChat please read our [documentation](https://zoitechat.readthedocs.org/en/latest/index.html):
- [Downloads](http://zoitechat.github.io/downloads.html)
- [FAQ](https://zoitechat.readthedocs.org/en/latest/faq.html)
- [Changelog](https://zoitechat.readthedocs.org/en/latest/changelog.html)
- [Python API](https://zoitechat.readthedocs.org/en/latest/script_python.html)
- [Perl API](https://zoitechat.readthedocs.org/en/latest/script_perl.html)
